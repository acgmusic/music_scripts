All Music by Falcom Sound Team jdk (Atsushi Shirakawa, Masaru Nakajima, Satoshi Arai, Hayato Sonoda, Kaname Ohara)
All Music Arrangement by Yukihiro Jindo

Violin: Ren Ohara
Guitar: Masaru Teramae
Bass: Atsushi Enomoto

This release is packaged together with the game. Apart from the soundtrack to Vantage Master Portable, it also features the original soundtrack for the first Vantage Master game on disc 2.

The PC version lists Atsushi Shirakawa, Masaru Nakajima, Satoshi Arai, Hayato Sonoda and Kaname Ohara as the composers, while the PSP version only lists Sonoda and Unisuga (even though it contains the exact same music). Only the arrangements from disc 1 -- which were handled by Yukihiro Jindo -- are different.
