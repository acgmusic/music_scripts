Notes

A special four-disc CD set celebrating Shiro SAGISU’s forty year anniversary of producing anime music. Featuring seventy tracks, including unreleased music, from the 1970s all the way to SAGISU's modern anime soundtracks.
Trivia: the album is set to be released on SAGISU's birthday (August 29).
