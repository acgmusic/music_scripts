QR Code inside the album leads to a Google Drive "#antique_the_united.zip" download, containing a single password protected .txt file titled "#antique_the_united.txt".
Each album contains a booster sleeve containing three collectible card, meant to be used for the album's ARG by using the morse code available across 14 different card on their bottom left. The decrypted .txt file contains a single URL which is used to go further within the ARG on a secret URL, with further more puzzles to decypher. Once the final password unlocked, an unlisted YouTube video becomes accessible, which is the song "Long Last the Child (Feat. AKA, eili)". The description contains yet another Google Drive archive containing the following:
-"Jackets" Folder
-"Card" Folder
-"Backgrounds" Folder
-"Antique -The United- mp3" Folder (containing the newfound bonus song, labeled as D4-M01)
-"appendix.pdf"
-"Appendix png.zip"
