Tracks without * or ** are covers loyal to their originals.
Tracks with * are enhanced covers done my own style.
Tracks with ** are original concepts inspired by different stuff.

---

Track Naming Legend:

0000 = Track Code Number
L = Track loops.
NL = Track does NOT loop. It's either the beginning or the ending to a loop track.
A / B / C / D ect. = Segments of a full track (usually the beginning "A", the loop part "B" and the ending "C"). Segments are noted right after Track Code Number.

---

Track Name Formulas:

Track with beginning, loop section and ending (BLE):
0001-A-NL
0001-B-L
0001-C-NL

Loop track:
0001-L

Non-Loop track:
0001-NL

*Note that segments of a BLE (Beginning-Loop-Ending) track are sold as individual tracks. - note that an intro followed by an endless loop combined into a single track will properly be played back only as OGG. Watch my tutorial on YouTube: youtu.be/OoyoQR5Sez4

*Note 2: This album also contains covers made from scratch, but no remixes. Cover and Remix are two different things.
